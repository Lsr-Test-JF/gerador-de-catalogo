# Gerador de Catálogo (GitHub Pages)

## Rodar local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Publicar no GitHub Pages

1. Faça push para a branch `main`.
2. No GitHub, habilite **Settings > Pages > Build and deployment > GitHub Actions**.
3. Acesse a URL publicada após o workflow `Deploy GitHub Pages` finalizar.
